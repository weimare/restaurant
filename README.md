# 🍽️ NOIR GOURMET — Сайт ресторана с доставкой и бронированием
## 📖 О проекте

**NOIR GOURMET** — это современный веб-сайт ресторана, объединяющий:

- 🍽️ **Меню** с категориями (холодное, горячее, закуски, десерты, напитки, сезонное, барная карта)
- 🛒 **Корзину** и оформление доставки (изменение количества, общая сумма)
- 📅 **Онлайн-бронирование** столиков
- 🌙 **Переключение** между тёмной и светлой темой
- 📱 **Адаптивную вёрстку** (ПК, планшеты, телефоны)
- 💾 **Сохранение** корзины и темы в браузере

**Концепция:** «Тёмная сторона вкуса» — премиальный стиль с золотыми акцентами, плавными анимациями и вниманием к деталям.

## 👥 Руководство пользователя (как пользоваться сайтом)

### 1. Навигация
В верхней части экрана — горизонтальное меню. Нажмите на любой раздел:

| Кнопка | Что открывает |
|--------|----------------|
| ❄️ Холодные | Холодные закуски и блюда |
| 🔥 Горячие | Основные горячие блюда |
| 🍢 Закуски | Лёгкие закуски |
| 🍰 Десерты | Сладкие блюда |
| 🧃 Безалкогольные | Напитки без алкоголя |
| 🌿 Сезонное | Блюда текущего сезона |
| 🥂 Барная карта | Коктейли, вина |
| 📅 Бронь | Бронирование столика |
| 🚚 Доставка | Корзина и оформление заказа |

### 2. Добавление в корзину
- **Впервые:** на карточке блюда нажмите **«➕ В корзину»**
- **Уже в корзине:** появятся кнопки **«−»** (убрать 1), **количество**, **«+»** (добавить 1)
- Количество можно менять прямо на карточке, без перехода в корзину

### 3. Иконка корзины (🛒)
- В правой части шапки отображается общее количество блюд
- Нажмите на неё — попадёте на страницу **«Доставка»** с полным заказом

### 4. Оформление доставки
На странице **«Доставка»**:
- Список блюд с кнопками **− / +** и ценой за позицию
- Общая сумма заказа
- Кнопка **«Очистить корзину»**
- Форма адреса (город, улица, дом, квартира)
- **«💳 Оплатить заказ (демо)»** — симуляция оформления (никаких реальных списаний)

### 5. Бронирование столика
- Перейдите в раздел **«Бронь»**
- Заполните: имя, телефон, количество гостей, дата/время, пожелания
- Нажмите **«Отправить бронь»** — менеджер свяжется с вами

### 6. Переключение темы
- Кнопка **🌙** / **☀️** рядом с корзиной
- Тема сохраняется в браузере

### 7. Социальные сети (демо)
В нижней части сайта — иконки соцсетей. В текущей версии ведут на заглушку `#`, но вы легко замените их на реальные ссылки.

## 🛠️ Руководство разработчика

### 1. Технологический стек

| Компонент | Технология |
|-----------|-------------|
| Разметка | HTML5 |
| Стилизация | CSS3 (Flexbox, Grid, CSS Variables) |
| Логика | JavaScript ES6 |
| Хранение | LocalStorage |
| Хостинг | GitHub Pages |

### 2. Архитектура

Весь проект — **один файл `index.html`**, содержащий:
- `<style>` — все стили (тёмная/светлая тема через CSS-переменные)
- HTML-разметку страниц (скрытые блоки с `display: none`)
- `<script>` — всю логику (SPA-навигация, корзина, формы)

Это сделано для простоты развёртывания и демонстрации.
