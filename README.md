# 🍽️ NOIR GOURMET — Сайт ресторана с доставкой и бронированием
## 📖 О проекте

**NOIR GOURMET** — это современный веб-сайт ресторана, объединяющий:

- 🍽️ **Меню** с категориями (холодное, горячее, закуски, десерты, напитки, сезонное, барная карта)
- 🛒 **Корзину** и оформление доставки (изменение количества, общая сумма)
- 📅 **Онлайн-бронирование** столиков
- 🌙 **Переключение** между тёмной и светлой темой
- 📱 **Адаптивную вёрстку** (ПК, планшеты, телефоны)
- 💾 **Сохранение** корзины и темы в браузере

**Концепция:** «Тёмная сторона вкуса» — премиальный стиль с золотыми акцентами, плавными анимациями и вниманием к деталям.

## 👥 Руководство пользователя (как пользоваться сайтом)

### 1. Навигация
В верхней части экрана — горизонтальное меню. Нажмите на любой раздел:

| Кнопка | Что открывает |
|--------|----------------|
| ❄️ Холодные | Холодные закуски и блюда |
| 🔥 Горячие | Основные горячие блюда |
| 🍢 Закуски | Лёгкие закуски |
| 🍰 Десерты | Сладкие блюда |
| 🧃 Безалкогольные | Напитки без алкоголя |
| 🌿 Сезонное | Блюда текущего сезона |
| 🥂 Барная карта | Коктейли, вина |
| 📅 Бронь | Бронирование столика |
| 🚚 Доставка | Корзина и оформление заказа |

### 2. Добавление в корзину
- **Впервые:** на карточке блюда нажмите **«➕ В корзину»**
- **Уже в корзине:** появятся кнопки **«−»** (убрать 1), **количество**, **«+»** (добавить 1)
- Количество можно менять прямо на карточке, без перехода в корзину

### 3. Иконка корзины (🛒)
- В правой части шапки отображается общее количество блюд
- Нажмите на неё — попадёте на страницу **«Доставка»** с полным заказом

### 4. Оформление доставки
На странице **«Доставка»**:
- Список блюд с кнопками **− / +** и ценой за позицию
- Общая сумма заказа
- Кнопка **«Очистить корзину»**
- Форма адреса (город, улица, дом, квартира)
- **«💳 Оплатить заказ (демо)»** — симуляция оформления (никаких реальных списаний)

### 5. Бронирование столика
- Перейдите в раздел **«Бронь»**
- Заполните: имя, телефон, количество гостей, дата/время, пожелания
- Нажмите **«Отправить бронь»** — менеджер свяжется с вами

### 6. Переключение темы
- Кнопка **🌙** / **☀️** рядом с корзиной
- Тема сохраняется в браузере

### 7. Социальные сети (демо)
В нижней части сайта — иконки соцсетей. В текущей версии ведут на заглушку `#`, но вы легко замените их на реальные ссылки.

## 🛠️ Руководство разработчика

### 1. Технологический стек

| Компонент | Технология |
|-----------|-------------|
| Разметка | HTML5 |
| Стилизация | CSS3 (Flexbox, Grid, CSS Variables) |
| Логика | JavaScript ES6 |
| Хранение | LocalStorage |
| Хостинг | GitHub Pages |

### 2. Архитектура

Весь проект — **один файл `index.html`**, содержащий:
- `<style>` — все стили (тёмная/светлая тема через CSS-переменные)
- HTML-разметку страниц (скрытые блоки с `display: none`)
- `<script>` — всю логику (SPA-навигация, корзина, формы)

Это сделано для простоты развёртывания и демонстрации.

### 3. Структура данных (меню)

```javascript
const products = {
    cold: [        // холодные блюда
        { id: 1, name: "Тартар из говядины", price: 1350, desc: "...", img: "URL", aspect: "landscape" },
        // ...
    ],
    hot: [ /* горячие блюда */ ],
    appetizers: [ /* закуски */ ],
    desserts: [ /* десерты */ ],
    soft: [ /* безалкогольные */ ],
    seasonal: [ /* сезонное */ ],
    bar: [ /* барная карта */ ]
};
Поля:

id — уникальный номер (должен быть уникальным во всех категориях)

name — название

price — цена в рублях

desc — описание

img — URL картинки (можно свой или через ImgBB)

aspect — landscape (16:9), square (1:1) или portrait (3:4)

4. Ключевые функции
Функция - Назначение
renderGrid(containerId, items) - Отрисовка карточек меню
renderAllCartButtons() - Обновление кнопок «+ / − / количество» на всех карточках
addToCart(product) - Добавить / увеличить количество
removeOneFromCart(id) - Уменьшить / удалить
renderCartPage() - Отрисовка корзины на странице доставки
updateCartUI() - Обновить счётчик на иконке корзины
activatePage(pageId) = Переключение между страницами (SPA)
setTheme(theme) - Переключение тёмной/светлой темы
5. Как добавить новое блюдо
Откройте index.html, найдите объект products

В нужной категории добавьте новый объект, соблюдая структуру

Присвойте новый уникальный id

Укажите ссылку на фото (лучше использовать ImgBB, Unsplash или свои файлы в папке images/)

Пример:

javascript
cold: [
    // ... существующие
    { id: 23, name: "Авторский тартар", price: 1490, desc: "С перепелиным яйцом", img: "images/new_dish.jpg", aspect: "landscape" }
]
6. Как добавить новую категорию меню
В products добавить новое свойство (например, breakfast: [])

В HTML добавить <div id="breakfast" class="page"><div class="section-title">...</div><div id="breakfastGrid" class="menu-grid"></div></div>

В объект pages добавить ссылку на этот элемент

В <ul class="nav-links"> добавить кнопку <button class="nav-btn" data-page="breakfast">🍳 Завтраки</button>

В initAllGrids() добавить renderGrid('breakfastGrid', products.breakfast)

В renderAllCartButtons() и findProductById() добавить 'breakfast' в цикл поиска

7. Изменение цветовой схемы
Цвета хранятся в CSS-переменных:

В блоке :root — тёмная тема

В блоке body.light-theme — светлая тема

Чтобы изменить акцентный (золотой) цвет — отредактируйте --accent-gold.

8. Хранение данных
Корзина: localStorage по ключу restaurant_cart

Тема: localStorage по ключу restaurant_theme

9. Развёртывание на GitHub Pages
=Создайте репозиторий на GitHub
=Загрузите в него файл index.html
=Перейдите в Settings → Pages
-В разделе Branch выберите main и папку / (root)
=Нажмите Save
-Через 1–3 минуты сайт будет доступен по адресу:
https://[ваш-логин].github.io/[имя-репозитория]/

10. Возможные проблемы
Проблема	Решение
Фото не отображаются - Проверьте, что ссылка на фото открывается в браузере. Используйте прямые ссылки (ImgBB, GitHub, Unsplash).
Корзина сбрасывается при очистке кэша - Это особенность localStorage. Для продакшена нужна серверная БД.
На мобильном меню не помещается - Горизонтальная прокрутка включена по умолчанию (класс .nav-wrapper).
